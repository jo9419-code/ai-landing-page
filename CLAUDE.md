# CLAUDE.md

이 프로젝트에서 작업할 때 반드시 따라야 하는 규칙입니다.

## 규칙

1. **정적 사이트만 사용** — API, 데이터베이스, 서버 사이드 로직을 추가하지 않는다. 폼 제출 등 서버가 필요한 기능은 `mailto:` 링크 등 정적인 방식으로만 처리한다.
2. **단일 파일 구성** — `index.html` 한 파일 안에 CSS와 JS를 모두 포함한다(인라인 `<style>`/`<script>`). 별도의 `.css`, `.js` 파일로 분리하지 않는다.
3. **한국어 콘텐츠** — 페이지에 노출되는 모든 텍스트(문구, 버튼, alt 텍스트 등)는 한국어로 작성한다.
4. **반응형 필수** — 모바일/태블릿/데스크톱 등 모든 화면 크기에서 정상적으로 표시되어야 한다.
